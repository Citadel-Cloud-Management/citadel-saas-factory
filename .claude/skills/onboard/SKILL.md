# Onboard Skill

## When to Invoke
- New developer joining the project
- Keywords: onboard, walkthrough, getting started, explain the project

## Walkthrough
1. **Architecture** — Clean layers, FastAPI backend, Next.js frontend
2. **Setup** — Clone, bootstrap.sh, docker-compose, environment variables
3. **Conventions** — Immutability, small files, TDD, conventional commits
4. **Agent System** — 265 agents across 15 domains, CYCLE_INTERVAL=0
5. **Development Workflow** — Branch, TDD, code review, PR, merge
6. **Deployment** — GitOps via ArgoCD, staging then production
7. **Key Files** — CLAUDE.md, _registry.yaml, docker-compose.yml, Makefile
