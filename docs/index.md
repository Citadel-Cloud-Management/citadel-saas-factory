# Citadel SaaS Factory

Universal Full-Stack SaaS Production Framework with 265 Autonomous Business Agents.

## Overview

Clone. Configure. Deploy. Any infrastructure. Zero software cost.

## Getting Started

See the [Quick Start](quickstart.md) guide to get up and running in minutes.

## Architecture

- **Backend**: FastAPI (Python 3.12)
- **Frontend**: Next.js 14 (TypeScript)
- **Database**: PostgreSQL 16
- **Infrastructure**: K3s + ArgoCD (GitOps)
