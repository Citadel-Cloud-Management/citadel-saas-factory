# Comparisons

Analysis pages born from research questions. When a query produces a valuable side-by-side analysis (tool A vs. tool B, pattern X vs. pattern Y, decision this way vs. that way), the answer is filed here so the next question that touches the same territory starts from the compiled comparison instead of re-deriving it.

## Page format

```markdown
---
name: <A> vs <B>
description: <one-line>
tags: [comparison, <domain>]
query: <original question>
date: YYYY-MM-DD
sources: [...]
confidence: 0.0 - 1.0
---

# <A> vs <B>

## Question
## Criteria
## Side-by-side
## Recommendation
## Sources
## Related pages
```
