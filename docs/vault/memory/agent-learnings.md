# Agent Learnings

Cross-agent knowledge base. Updated as agents discover patterns and solutions.

## Patterns
- Use immutable data structures across all agents
- Prefer composition over inheritance
- Each agent should be stateless — persist state via memory/ files

## Integration Notes
- Ruflo orchestrates multi-agent workflows via MCP
- Graphify provides codebase knowledge graph for navigation
- Agents consult the graph before making changes
